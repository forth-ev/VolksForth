\\ savesystem                                            11Nov86

Dieses File enthaelt das Utility-Wort SAVESYSTEM.

Mit ihm kann man das gesamte System als File auf Disk schreiben.

Achtung:
   Es wird SAVE ausgefuehrt, daher ist nach SAVESYSTEM
   der Heap geloescht!

Benutzung:     SAVESYSTEM <filename>
